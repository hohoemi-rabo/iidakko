## 推奨着手順

1. **01 共通基盤**（5タブ・テーマ・型）→ 土台を固める
2. **08 データ整備**（先に数件だけでも入れると画面実装が進む）
3. **02 共通コンポーネント** → 各画面で再利用
4. **03〜07 各画面**（主役の **05 マップ** を優先度高で）
5. **09 ビルド・配布**（マップ実機確認のため EAS 開発ビルドが必要）
