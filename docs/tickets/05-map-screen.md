# 05 — マップ画面（map）★アプリの主役

登録不要の子育て施設マップ。カテゴリ色ピンで施設を地図表示し、ピンタップで施設カードがせり出す。地図⇄一覧のトグルとカテゴリ絞り込みを備える。

## 参照
- `REQUIREMENTS.md` 5章③マップ・6章（Facility 型）・3章（地図データの作り方）
- `docs/ui-references/map.png`（このチケットの見た目の基準）

## 依存
- 01（基盤）, 02（共通コンポーネント）, 08（施設データ＋座標）, 09（Maps APIキー・開発ビルド）

## 前提・注意
- **`react-native-maps` は Expo Go では動かない。** config plugin で導入し、**EAS 開発ビルド**で確認する（→ 09）。
- **実行時ジオコーディングはしない。** 座標は 08 でビルド前に確定し `facilities.ts` に数値で直書き。
